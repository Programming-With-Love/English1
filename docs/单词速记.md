## 5个元音: 地位卑微

元音字母可以任意的改变: (而不改变单词意思)

1. 互换位置
2. 加减字母

sit set seat

### a 字母

a / a+辅音 开头=> 语气词，一般是加强语气
  => accelerate 加速  ac 加强语气 + celer 噻啦，马奔跑的声音 + ate 动词后缀
  => abrupt 突然的、唐突的、无理的
       rupt 爆炸

af => 加强语气
    => affluend 富裕的(富得流油的)  af(加强语气) + flu(流) + ent(形容词后缀)


alt => 高  (来源 tall)
    => altitude 高度 alt + itude 


ami => 爱  （来源于丘比特的拉丁文名字）
    => amiable 可爱的、和蔼的


angu => 苦恼  （angry 生气）
    => anguish 极度痛苦  angu + ish


### e 字母

e / ex / e+辅音 开头 => 向外
    => exit 出口、推出
    => export 港口
    => expose 暴露 v（把 姿势/造型 展现出来）
    => exposure 暴露、被报道 n
    => erupt （火山）向外 + 爆发
        rupt 爆炸
    => extra 额外
      extraordinary 不平凡、极端的、不可思议的
      extreme 极其的、非常的


### i 字母


in  => 进入、使动(进入某中状态)、否定(进入某种状态，进入另一种极端，反义同源)、加强
    => in/ir/il/i bmp => im 
en => 进入、使动(进入某中状态)
un => 否定(进入某种状态，进入另一种极端，反义同源)

i + dent 牙齿  + ity => 牙齿意味着身份（奴隶社会的买卖，看牙齿就知道健康程度）


## 21个辅音: 满足通假（d=t 读音类似、意思就类似）

### b=p 字母

b => 木头：一般木棒就是上下不规整的
    => bar 小木棒
    => ban 禁止

bar: 王磊
car: 李磊
far: 孙磊

mother -> father -> brother

* 类似汉语起名字: 性不同，名一样
* b/c/f 代表意思
* ar 代表后缀


br => 一分为二，一分为多
    => brook 支流、消息          (cook、book)
    => branch 树枝、分支         (lunch、bench)
    => brain 大脑（内部很多分支）  (gain、rain)
    => bruch 刷、刷子            (rush 冲)
    => break 打碎               (weak 虚弱、弱的)

bo => 崩，爆炸的意思
    => bomb 爆破的声音

bread 面包、钱

aboard 在(船、飞机、公共汽车、火车等)上


abroad 广阔的、出国
 broadcast 广播  broad 广 + cast 信号


=> ballot 小球，选票
    -> white ballot 好球
    -> black ballot 坏球



### c=k 字母

c => 弯曲、包含、抓（可以用来保护自己）
  => cave 山洞

ca => 包裹、抓、运输
  => cave 山洞
  => camp -> comp 梳头的感觉（用手去抓）
  => camp 野营
  => camel 骆驼（驼峰和头之间的形状 C） -> l 清辅音，由 came + l 
  => car 车子，运输工具
  => cabbage 卷心菜  
        page 页 
        wage 薪水
        cage 鸟笼

cap => 鸭舌帽
    => capital 首都，用帽子罩着地方 it al 两个字母的合成结尾，类似 黄晓、黄明、你就造词 黄晓明
    => captain 队长、船长、老大，用帽子罩着大家
          pain 疼痛
          gain 获得
        remain 保持、保留
        obtain 获得

cat -> kat (来源 kitty 猫 - 元音字母可以自由转换)


cl => 合成
   => close 关门
       pose
   => closet 小隔间（关人的小隔间） it、et、ot 结尾表示小的意思
      toilet 厕所，排油的小隔间 oil 油 oily 油的
     blanket 毛毯，覆盖（来源于 小白布，不吉利）blank 空白  blac 白色 <-> black 黑色（反义同源）
     deposit 沉淀物、存款（每个月的沉淀物，余款） （dig 向下、deep 深、down 下降 => 向下）
   => class 班级，由很多人合成的
   => club 俱乐部，由很多人合成的
   => clap 鼓掌，把两个手合在一起

cand => 白色
     => candle 蜡烛（白色的）
     => candy  糖果（以前是白色）
        candi ... ate(动词后缀) (连读 两个元音发音不好听)
        candidate (加 d 读起来好发音)
     => candidate 候选人
         graduate 毕业、毕业生

cess
  => access  行走; 走路; 前进; 接近入口; 通道; 通路
     process 前进; 过程; 程序
     accessible 




co / con / com / co+辅音  => 一起

### f 字母

fine => 好到最优程度，就开始不好了
    => 好的意思
    => 罚款的意思
final/finally => 好到最后就结束了、最终的



### n=r=l (清辅音)

由于误写造成的，例如：莎士比亚当年写的时候，羽毛慢慢没水了

n -> r -> l  (慢慢羽毛没水、字慢慢写不显了)

man -> male <=> female (反义同源)

rose => 罗丽丝 （r 和 l 类似）

set -> sert (口音问题造成的误读， r是清辅音，很容易误插入到单词中)
  => insert 向里放置，插入


present 提前到了，出席
 absent  ab否定前缀， sent -> sert -> set 坐的意思，整体就是：未坐，缺席

> l => 长的意思

l => long 长的意思
  => line 线条

> r 字母

re => 再一次
   => rebuild 重建
   => review  检查
   => return  返回
   => reply 回复


rupt => 爆破
     => ripe 熟的、爆破
     => erupt 向外爆炸、火山爆发
     => abrupt 加强爆破的状态、突然的、唐突的、无理的
     => bankrupt 银行爆炸了，就是破产

### p 单词

pre / pro 往前，前面
  => history 史前
  => president 坐在最前面的人（校长、主席、总统、董事长、帮主）
  => predict 预言   （pre 往前 dict 说）


pan => 胖有关
    => panda 大熊猫
    => expand 膨胀 v（肚皮不断向外膨胀的过程）
       expansion 膨胀 n


ped = pad 板、平板
  => expedition 远征，探险  ex + ped + ition


pens
  => spend 花费、花钱 v
  => expensive 昂贵的 ex + pens + ive 把钱花出去
  => expense 花费 n 
  => pension 抚恤金，补贴 pens + ion


plic => 折叠
     => complicated 错中复杂 com + plic + ated 一起交错在一起
     => explicit 明确的，清晰的 ex + plic + it 把折叠的东西打开，不隐藏


pl => 说
   => explore 探索 ex + pl + ore 向外说，告知你的过程，叫做探索


### s 单词

  soup 汤（喝汤的声音） 相声词
  sorb 吸进去(喝汤的动作) b和p 声音相似
absorb 吸引（用力吸进去）


solo => 唯一、独一无二、独奏
     => sol = sun
     => absolute  绝对的、第一无二的  ab 加强语气 + sol + ute(ate 动词/名词后缀)

str => 拉长、绷紧   
    => street 街道
    => stream 小河; 溪
    => struggle 挣扎
    => strong  肌肉
    => abstract 抽象的（拉长的事物，就变抽象了）

spy => 间谍
    => inspect  检查 in + sp + ect     往里看
    => prospect 前景 pro + sp + ect    往前看
       prophet  预言 pro + ph + et     往前说
    => suspect  怀疑 su(b) + sp + ect  往下看
    => expect   期望/预期 v  ex + sp + ect 往外看（看出来） (ex 和 s 发音重合，就去掉 s)
       expectation  期望/预期 n 

sc =>  c = sc 雕刻的意思
    => caver 雕刻
    => scale 刻度、鱼鳞、去掉鱼鳞
    => scandal  丑闻（挖到了缺点） scar 疤痕（缺陷） + and + al 合成结尾
    => scarf 围巾（脖子上雕刻了一个东西） sc + arf
    => scoop 铲子、勺子、挖新闻  sc(雕刻，在土里，就是铲子；在媒体，就是挖新闻) + oop
    => sculpture  雕刻、雕像  sc + ul + pt + ure 合成结尾（类似： 黄晓、黄明、黄晓明）
    => score 分数、得到分数  sc + ore



sn => 势力
    => snoopy 史努比，狗！ 像狗一样势力、爱探听的、爱管闲事的、讥讽和嘲笑
    => snobbish 势力的，自命不凡的
    => sneer 嘲笑、讥讽
    => sniff 感冒了，用力吸气
    => snore 打鼾  sn + ore
